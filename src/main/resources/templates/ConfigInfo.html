Get started with Confluence Exporter

The Confluence Exporter plugin will help to export your pages from any Confluence project in a document format with just few clicks.


  • Currently supported documents are:
    1. PDF

    2. WORD (.doc)

  • How to export the pages:

    1. After installing the plugin and adding appropriate license to it, The Confluence exporter option gets added to the right section of opened issue.

    2. The next step is to click on miniOrange PDF exporter option.

    3. Pop-up will get open. Then select the fields which you want to export from the Confluence page.

    4. Then select document format in which you want to export the Confluence page.

    5. Click on Download to export the page in document format
